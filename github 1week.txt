깃, 깃허브 1주차 수업 정리 

깃과 깃헙의 차이점

- 깃은 내 컴퓨터에 사용하고 버전관리, 협업 관리를 위한 도구
- 깃헙은 깃 프로젝트를 원격으로 올릴 수 있는 저장소기능, 커뮤니티기능 
- 소스트리는 깃을 쉽게 사용할 수 있는 도구

버전관리란 - 누가,언제,현재 프로젝트의 내용이 어떤지 정보를 남기는것

깃의 버전관리하는 방법 - 커밋을 해서 현재 프로젝트를 저장한다.(스냅샷)

소스트리에 설정에 들어가서 계정이 잘 연결되어 있는지 확인해야 한다.

Command + Shift + .(점) =  . 으로 시작하는 파일 숨기기, 드러내기 단축키

git initialize - 소스트리에 파일을 추가하는것 

트래킹 - 깃도 원격 repo 와 로컬 repo 을 연결해서 사용

클론 - 원격 repo 을 로컬 repo 에 복사해서 내려받는것 

로컬 repo 에 어떤 원격 repo 가 연결되어 있는지 정보가 있다.
( 작업내용을 자동으로 커밋하지는 않는다. ) - 수동반영 해야함

push, pull 은 로컬 리퍼지토리 기준이다.

오류 - sourcetree  한국어로 할때 창이 오류났는데 영어로 바꾸니깐 됨
